🏗️ Account Structure 101

How to Organize Campaigns for Scale and Sanity

⏱️ 50 Min
🟢 Beginner

Why Account Structure Matters

Bad structure = impossible to optimize. Good structure = easy to scale.

The Problem:

You have 100 campaigns, 500 ad sets, 2000 ads. You can't find anything. You don't know which creative works. Optimization is chaos.

The Solution:

Organize by: Objective → Audience → Creative → Hook. Each level has a purpose. Everything is findable.

The 3 Levels of Organization

Level Purpose Example
Campaign Business Objective Awareness, Consideration, Conversion
Ad Set Audience + Budget Cold Audience (18–35), Warm Audience (Visitors)
Ad Creative Variations Hook #1, Hook #2, Hook #3

🟢 Simple Structure (Start Here)

If you're new or have < 3 products:

📊 Campaign: "Product-Name_Conversion"
└─ Ad Set 1: "Cold Audience"
├─ Ad: "Hook #1"
├─ Ad: "Hook #2"
└─ Ad: "Hook #3"
└─ Ad Set 2: "Warm Audience"
├─ Ad: "Testimonial #1"
└─ Ad: "Case Study"

That's it. 1 Campaign. 2 Ad Sets. 5 Ads.

Everything fits in your head. You can manage it. When one grows, you duplicate and split.

🚀 Advanced Structure (Scale Here)

If you have 5+ products or $50k+ monthly budget:

📊 Campaign Group 1: "Product-A_TopOfFunnel"
├─ Campaign: "Product-A_Awareness_Content"
├─ Campaign: "Product-A_Awareness_Video"
└─ Campaign: "Product-A_Awareness_UGC"
📊 Campaign Group 2: "Product-A_Conversion"
├─ Campaign: "Product-A_Conv_Cold_20–45"
├─ Ad Set: "Interest-Targeting"
├─ Ad: "Hook_Pattern-Interrupt"
├─ Ad: "Hook_Curiosity"
└─ Ad Set: "Lookalike-2%"
└─ Campaign: "Product-A_Conv_Warm_Retargeting"
├─ Ad Set: "Visitors_Last-7d"
└─ Ad Set: "Visitors_Last-30d"

Why This Structure Works at Scale

  • Top-Level: Easy to see which Objective is winning (Awareness vs Conversion vs Retention)
  • Campaign-Level: Easy to see which Format works (Content vs Video vs UGC)
  • Ad Set-Level: Easy to see which Audience segment is profitable
  • Ad-Level: Easy to see which Hook works

Result: When performance drops, you know exactly WHY and WHERE.

📝 Naming Conventions That Save Your Life

Bad names = chaos. Good names = instant clarity.

Campaign Naming

❌ "New Campaign" / "Test 1" / "Facebook Ads"

Too vague. In 2 weeks you won't remember what this is.

✅ "ACME-ProductX-Conversion-ColdAud-20250110"

Brand + Product + Objective + Audience + Date

Formula:

[BRAND]_[PRODUCT]_[OBJECTIVE]_[AUDIENCE]_[DATE]

Example:

ACME_Shoes_Conv_Cold-18-25_20250110

Ad Set Naming

Formula:

[AUDIENCE-TYPE]_[AGE-RANGE]_[BUDGET-DAILY]

Examples:

Cold-Interest_18-35_50EUR
Warm-Visitor_Alle_100EUR
Hot-Retarget_Last7d_200EUR

Ad Naming

Formula:

[HOOK-TYPE]_[FORMAT]_[VARIATION]

Examples:

PatternInt_Video_V1
Curiosity_Carousel_V2
Social-Proof_Static_V3

🧪 Testing vs BAU: Keep Them Separate

This is critical: Never mix testing with scaling.

🧪 Testing Campaigns (Separate)

Name with "[TEST]" prefix. Keep budget small ($10–30/day). Run for 5–7 days only.

[TEST]_NewHook_V1_20250110

Purpose: Learn which hook/audience/creative wins

📈 BAU Campaigns (Scaling)

Name WITHOUT [TEST]. These are your profit makers. Scale budget slowly.

ACME_Shoes_Conv_Cold_20250110

Purpose: Generate revenue consistently

Weekly Test Strategy

  • Week 1: Test Hook #1 vs Hook #2 (small budget, 7 days)
  • Winner: If Hook #1 wins → move to BAU campaign, scale budget
  • Week 2: Test new Audience Segment vs Control (same structure)
  • Loser: If Hook #1 loses → pause test, learn why, try different hook next week

Benefit: BAU stays stable + predictable. Testing finds next winners.
